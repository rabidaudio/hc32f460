平台说明
================================================================================
GCC工程，由Eclipse IDE外挂GNU-ARM Toolchain，再结合pyOCD GDB Server实现工程的编译、
链接和调试。在用Eclipse导入工程后，请将xxxx_PyOCDDebug中pyocd-gdbserver和SVD文件
设置为正确的路径；请将xxxx_PyOCDDownload中pyocd设置为正确的路径。注意，这些路径不
能包含非英文字符。


功能描述
================================================================================
本样例主要展示MCU的TIMER6的PWM的三角波模式、双缓存输出功能，
说明：
配置timer6，hclk设置为168M，周期400us

================================================================================

---------------------

辅助工具:
---------------------
示波器

辅助软件:
---------------------
无

================================================================================
使用步骤
================================================================================
1）  打开工程编译并全速运行。
2） 使用示波器测试PE09和PE08的输出波形，看到PE09(Timer61 PWMA)和PE08（Timer61 PWMB）输出PWM波形,
PWMA和PWMB的3种占空比交替变化
